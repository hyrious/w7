## @hyrious/w7

This is a small **subset** of [ulivz/w7](https://github.com/ulivz/w7).

**Note**: This library is ESM only.

### Usage

```bash
npm i -g @hyrious/w7
w7 index.html
```

### Alternatives

- [w7](https://github.com/ulivz/w7)
- [sirv](https://github.com/lukeed/sirv)
- [vite](https://vitejs.dev)
